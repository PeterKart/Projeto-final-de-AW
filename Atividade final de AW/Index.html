Projetos de Autoria Web

Avaliação - Site com Bootstrap

Nesse projeto utilizei o framework Bootstrap para a melhorar o design do site. Foi dada a proposta para criação de 3 páginas linkadas com o tema de algumamarca famosa, escolhi a marca Spotify.



Atividade - Site com três colunas

Nesse projeto fiz um site com 3 colunas falando um pouco sobre a história da minha cidade. Fiz 1 página contando a história da cidade, 1 página como a galeria de fotos que representa a cidade e 1 página Home contando as principais notícias da cidade.



Atividade - Site com o tema blog

Nesse projeto fui instruído a fazer um site com 3 páginas contendo: Logotipo, menu, Imagem de capa, Área de postagens com 5 postagens(todas com fotos), Barra lateral(postagens recentes e Categorias) e Rodapé. Peguei como exemplo para o blog a minha antiga escola e coloquei algumas postagens sobre eventos que aconteceram.
